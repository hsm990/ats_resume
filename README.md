# ATS Resume Builder & Analyzer

An AI-Powered ATS (Applicant Tracking System) Resume Analyzer and Builder. This application helps users create professional, ATS-friendly resumes and analyze them to improve their chances of passing automated screening systems.

## Features
- **Resume Builder**: Intuitive interface to build resumes with various sections (Experience, Education, Skills, etc.)
- **ATS Optimization**: Guidelines and structural formatting to ensure resumes are easily parsed by ATS.
- **AI Integration**: AI-driven suggestions and analysis (using Gemini API) to enhance resume content.
- **Modern UI**: Built with React, Tailwind CSS, and Material UI for a polished user experience.

## Tech Stack
- React 19
- Vite
- Material UI
- Emotion & Styled Components

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run the development server**:
   ```bash
   npm run dev
   ```
3. **Build for production**:
   ```bash
   npm run build
   ```
